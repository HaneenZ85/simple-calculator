# 🧮 OOP Calculator in C++

A simple console-based calculator built using C++ and Object-Oriented Programming (OOP).

---

## 💡 Project Description
This project is a menu-driven calculator that performs basic arithmetic operations using a `Calculator` class.

The user can repeatedly choose operations until exiting the program.

---

## ⚙️ Features
- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division (with zero division handling)
- 🔁 Continuous menu using loop
- ❌ Input validation for invalid choices

---

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- Classes & Objects
- Functions (methods inside class)
- Conditional statements (if / else)
- Switch-case
- Loops (do-while, while)
- Input validation

---

## 🚀 How It Works
1. Program displays a menu of operations
2. User selects an operation (1–5)
3. User enters two numbers
4. Result is displayed using class methods
5. Menu repeats until user chooses Exit

---

## 📌 Example Flow
Choose operation → Enter numbers → Get result → Repeat → Exit
